# SolutionSecurePassword

Português
 Sobre
Esta é a minha solução para o desafio Senha Segura do repositório backend-br/desafios.
 Descrição do Desafio
Criar um validador de senhas que verifica se uma senha atende aos requisitos de segurança. Uma senha segura deve:

Ter no mínimo 8 caracteres
Conter pelo menos uma letra maiúscula
Conter pelo menos uma letra minúscula
Conter pelo menos um número
Conter pelo menos um caractere especial (!@#$%^&*()-+)

 Tecnologias Utilizadas

Java
Spring Boot

 Como Executar
bash# Clone o repositório
git clone https://github.com/sheylaghost/secure-password-solution.git

# Entre no diretório do projeto
cd secure-password-solution

# Execute com Maven
mvn spring-boot:run

English

 About
This is my solution to the Secure Password challenge from the backend-br/desafios repository.
 Challenge Description
Create a password validator that checks if a password meets security requirements. A secure password must:

Have at least 8 characters
Contain at least one uppercase letter
Contain at least one lowercase letter
Contain at least one number
Contain at least one special character (!@#$%^&*()-+)

 Technologies Used

Java
Spring Boot

 How to Run
bash# Clone the repository
git clone https://github.com/sheylaghost/secure-password-solution.git

# Navigate to the project directory
cd secure-password-solution

# Run with Maven
mvn spring-boot:run
